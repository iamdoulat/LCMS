# Sound Files

This directory contains sound effects for the application.

## Required Files

- `water-drop.mp3` - Water drop sound effect for pull-to-refresh action

Please add a water drop sound file (mp3 format) to this directory.
You can find free sound effects from:
- https://pixabay.com/sound-effects/
- https://freesound.org/
- https://mixkit.co/free-sound-effects/

The sound should be:
- Short (< 1 second)
- Pleasant water drop sound
- MP3 format
- Named: water-drop.mp3
